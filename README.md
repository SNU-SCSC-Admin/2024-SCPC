# 제2회 SCPC(SCSC computer programming contest)
![image](https://github.com/SNU-SCSC-Admin/2024-SCPC/assets/31611070/0ae5d775-f7f9-41a2-a9a1-7c426c239d3b)

## 소개

서울대학교 SCSC 동아리에서 제2회 SCPC(SCSC computer programming contest)를 개최합니다!

알고리즘 PS를 좋아하는 누구나 참가할 수 있는 프로그래밍 대회입니다.

알고리즘, 자료구조, 수학 등 전반적인 분야를 다루는 icpc 스타일의 프로그래밍 문제 `8`문제가 주어지고, `3`시간동안 진행됩니다.

실력자를 위한 **div1**과 초심자를 위한 **div2**로 분리하여 경쟁합니다. **div1**은 백준 solved AC 기준 `Silver` ~ `Ruby`, **div2**는 `Bronze` ~ `Diamond` 수준의 문제가 나옵니다.

```
일시: 5/25(토) 오후 2시~6시 (대회는 5시까지만 진행됩니다.)

장소: 추후 공지 예정 (서울대학교 내에서 진행됩니다.)

참여 대상: 대학생 및 대학원생

참가비: 무료
```

### 상금 및 혜택
- 200만 원 상당의 상품 제공
- 1등: 갤럭시 탭 / 2~3등: 갤럭시 버즈 (div1 기준)
- 백준 배경화면 및 배지 제공
- 기념 굿즈 제공
- 후원사 (현대 모비스, 로그프레소) 채용 홍보


외부인 참가 링크: https://docs.google.com/forms/d/1ljMJFznI6ktLZxaHMBykfXeekl2uZGsqGRAclfNwgEA/edit
